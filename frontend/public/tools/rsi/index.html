Inside the Plugin

RSI Workflow

Analyze residential layouts, diagnose inefficiencies, compare schemes, and evaluate financial outcomes directly in Revit.

Capabilities

Built for Early-Stage Feasibility

Live Efficiency Analysis

Calculate Gross-to-Net ratio, core proportion, and circulation benchmarks in real time from live Revit model areas.

Heatmap Diagnostics

Colour-coded overlays highlight inefficient zones — oversized cores, wasted circulation, and underperforming layouts at a glance.

Scenario Comparison

Save snapshots of alternative design options and compare key metrics side by side to identify the strongest scheme.

Financial Impact Estimation

Estimate revenue effects of layout changes with built-in cost and value modelling tied directly to scheme performance.

Exportable Reports

Generate formatted PDF reports, multi-sheet Excel workbooks, and raw CSV exports ready for client presentations or internal review.

Native Revit Integration

Runs as a dockable panel inside Autodesk Revit — no external tools, no file round-tripping, no context switching.
